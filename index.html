Binary Search

The Binary Search Algorithm takes a sorted array of items and searches for a desired element in O(log(N)) time where N is the size of the array. In particular, it keeps track of a lower, middle, and upper position in the array. This allows the search algorithm to discard half of the current search area after every iteration, hence why it's time complexity is in O(log(N)).
